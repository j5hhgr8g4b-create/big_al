# Big Al — Build Plan v1

Build milestone by milestone.

## Milestone 0 — Project Setup
Next.js TypeScript app, Tailwind, Supabase client, env vars, app shell, bottom nav.

## Milestone 1 — Auth + Restaurants
Supabase Auth, profiles, chefs, restaurants, restaurant_members, auto-create Chef profile, create Restaurant flow.

## Milestone 2 — Cookbook + Recipes
cookbooks, recipes, ingredients, recipe_ingredients, recipe_steps, create/edit/view/archive recipe.

## Milestone 3 — Imports
imports table, manual import form, URL/text fields, parser placeholder, convert import to recipe, Needs Review.

## Milestone 4 — Recipe Books + Search
recipe_books, recipe_book_recipes, create/edit book, add recipe to book, search by title and ingredient.

## Milestone 5 — Menu
meal_events, This Week, Next Week, Unplanned, add recipe to date, people eating.

## Milestone 6 — Shopping
shopping_lists, shopping_items, generate from planned meals, consolidate, tick off, manual add.

## Milestone 7 — Cook Mode
step-by-step route, Previous/Next, timer, mark cooked, cook again yes/no.

## Milestone 8 — Basic Big Al
contextual help panel, search recipes, recommend recipes, answer simple recipe questions.
