# Codex Instructions for Big Al

Big Al is a living cookbook designed to make cooking easy and enjoyable.

Read these files before changing code:

1. SAFE_CODEX_GITHUB_SETUP.md
2. CODEX_RULES.md
3. docs/PROJECT_GOVERNANCE.md
4. PRODUCT_SPEC.md
5. DATABASE_SCHEMA.md
6. APP_SCREENS.md
7. BUILD_PLAN.md
8. docs/CURRENT_STATUS.md

Build one milestone at a time.

The MVP loop is Cookbook to Menu to Shopping to Cook Mode.

Keep the app mobile-first, simple, practical, and recipe-centred.

Do not add social media mechanics, grocery price comparison, calorie tracking, or a generic AI chat app during the MVP.

After each task, report:
- what changed
- files changed
- database changes
- checks run
- audit docs updated
- what Alex should test next
